Cara publish Google Sheets sebagai CSV:
  1. Buka Google Sheets → File → Share → Publish to web
  2. Pilih sheet/tab yang berisi data report DOT
  3. Pada pilihan format, pilih Comma-separated values (.csv)
  4. Klik Publish, lalu copy link yang muncul
  5. Tempel link tersebut di kotak di atas, lalu klik Sambungkan & Muat Data

Catatan: setiap kali sheet diubah dan disimpan, data publish biasanya update otomatis dalam beberapa menit. Tinggal klik "Muat Data" lagi untuk refresh.
